# Customer-Behavior-dashboard
 Customer Behavior Dashboard to analyze customer purchasing patterns, spending habits, and engagement trends using SQL, Python, and Power BI. The project involved data cleaning, preprocessing, and analysis of customer transaction data to generate meaningful business insights. 

Used 
* SQL for data extraction, transformation, and querying large datasets.
* Python was utilized for data preprocessing, exploratory data analysis (EDA), and statistical analysis using libraries like Pandas and Matplotlib.
* Built dynamic and visually interactive dashboards in Power BI.

The dashboard helps businesses understand customer behavior, identify high-value customers, improve marketing strategies, and support data-driven decision-making through real-time visual insights.
